Changelog

Histórico de mudanças do Swap

v1.1.0 13 de Maio de 2026

Adicionado

  • Conversão de eBooks (EPUB, MOBI, AZW3, FB2, LRF, RB, TCR, SNB, PDB)
  • Conversão de apresentações (PPT, PPTX, ODP, PPS, PPSX, PPTM, POT, POTX, POTM, PPSM)
  • Conversão de vetores (SVG → PNG, PDF, JPG, WEBP)
  • Conversão de quadrinhos (CBZ, CBR, CB7 → PDF, CBZ)
  • Upload via URL (converter arquivo diretamente de um link)
  • Acessibilidade com ARIA labels e navegação por teclado
  • Páginas Changelog, Roadmap e Sobre no menu
  • Checkboxes visuais no Roadmap (☑/☐)
  • Informações da organização AFSIS na página Sobre

Corrigido

  • Erro na conversão XLSX → PDF via LibreOffice
  • Alinhamento e espaçamento uniforme nos componentes da interface
v1.0.0 12 de Maio de 2026

Adicionado

  • Conversão de imagens raster (JPG, PNG, WEBP, BMP, TIFF, ICO) via Pillow
  • Conversão SVG → PNG/PDF via cairosvg
  • Conversão PDF → PNG (por página), DOCX, TXT via PyMuPDF e pdf2docx
  • Conversão de documentos (DOCX, ODT, TXT, RTF, HTML) via python-docx e LibreOffice
  • Conversão de planilhas (XLSX, CSV, ODS) via openpyxl e pandas
  • Drag & drop de arquivos na interface
  • Conversão em lote com ThreadPoolExecutor
  • Barra de progresso em tempo real via WebSocket
  • Histórico de conversões com persistência JSON
  • Configurações por formato (qualidade, DPI, redimensionamento)
  • Tema claro/escuro com persistência
  • Interface web completa (FastAPI + Vanilla JS)
  • Localização pt-BR
  • Testes unitários com pytest

Sobre o Swap

Conversor de arquivos universal

Versão 1.1.0
Licença MIT
Organização AFSIS @ GitHub
Repositório afsis-dev/Swap
Backend Python 3.11 + FastAPI
Frontend Vanilla JS + CSS3 + HTML5
Formatos 50+ suportados
Histórico ~/.fileconverter/history.json

O Swap nasceu da necessidade de converter arquivos entre formatos sem depender de serviços online que comprometem a privacidade. Totalmente offline, processa tudo localmente — seus arquivos nunca saem da sua máquina.

Projeto mantido pela organização AFSIS em github.com/afsis-dev/Swap
